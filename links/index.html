Other Good Browsers

Mozilla

A complete internet suite with browser, email client, newsgroup client, IRC chat client, and HTML editing. Based on the gecko engine which is also used on Netscape Navigator and Firefox.

Opera

An extremely powerful, highly customisable and standards complaint browser. Features a lot of tools and extras for power users including mouse gestures and a popup stopper. It also comes with integrated mail, news and chat clients. The software is adware, which means it display adverts on the browser unless paid for.

Netscape Navigator

A well known rebranding of the original Mozilla Browser. Also based upon the gecko engine, this browser was Internet Explorers main competitor in the '90s.

K-Melon

An extremely fast, customisable and lightweight browser for Windows platform. It is open source software, and uses the Gecko rendering engine.

Konqueror

A lightweight, standards compliant web browser for the *nix operating system.

Safari

This browser is the Mac OS X default web browser.
